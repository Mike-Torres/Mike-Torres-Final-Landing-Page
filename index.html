Why do I love being a Developer?

When I found out that ninja's weren't a thing anymore, and cavemen weren't taken seriously, I had to go to my backup plan. I figured, I was pretty good in math and wanted to learn to build things. When I was younger, I loved building legos into awesome 3-D Super Mario Characters and when I found out that I could create the same type of projects by coding I was sold. It started when I created Tetris, but not any old Tetris in JavaScript. I built a Tetris game that ran within the GameBoy I created. I figured if you're going to build an old school game, you have to build it the right way!

The main reason why I love this is because there is so much to learn in software programming. There will never be a day as a programmer where I will ever say I know it all and for me to be able to continually learn new technologies is one of the funnest things anyone can do.
